# インストール
1.最新リリースをダウンロードして解凍します<br>2.ゲームのフォルダーを見つけます<br>3.ゲーム実行ファイルがあるディレクトリに解凍したフォルダの中身を全てペーストします<br>4.ゲームを実行します
